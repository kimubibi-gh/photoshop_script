#WIN11用_リサイズ&リネーム

Photoshopを使わず、リサイズとリネームをします。

注意事項

Win11用です。powershellを使用しています。

jpgのクオリティは下げてません。

----------------------------------------------

★まずやること（さいしょだけ）

・このファイルが入っているフォルダごとデスクトップに保存。

----------------------------------------------

▼作業内容
1.処理フォルダの下準備をする。
　1-1.フォルダ名の商品番号の前にスペースが入って居なかったら入れる。
　1-2.フォルダの中身を見ていらないものを削除する（タグなど）。
　1-3.順番は名前順なので、カウント順序を変えたい場合は頭に数字を入れるなどして調整する。

2.★RenameResize1000.vbsにフォルダをドラッグドロップ（複数同時可能）

3. 「リサイズ済み」フォルダに保存
